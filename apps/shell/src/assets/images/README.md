# School Building Image

Place a high-quality modern school building image here named `school-building.jpg`.

Recommended specifications:
- Format: JPG or PNG
- Resolution: 1920x1080 or higher
- Style: Modern school building with visible architecture, greenery, welcoming appearance
- The image will have a blue gradient overlay applied via CSS

Example sources:
- Unsplash: https://unsplash.com/s/photos/modern-school-building
- Pexels: https://www.pexels.com/search/modern%20school/

If no image is provided, a blue gradient background will be displayed instead.
